SPARQL-endpoint

General information

This page gives you access to query our PTW dataset. All the neccessery prefixes are allready predifined. The visualisation is made possible by sgvizler. You can select the endpoint you want to use by editing the Endpoint-field below to point to the correct service. The url is on the form 

http://fuseki.fone.no:3030/DATASET/SERVICE
Available services and datasets are listed to the right.

The default query will just fetch 20 random triples from any available graph

SELECT ?subject ?predicate ?object ?g 
WHERE { GRAPH ?g 
        {?subject ?predicate ?object}
} LIMIT 20

You can also select a chart type, but the query must correspond to the requirments set by the given chart. You can views the requirements here.

Please be patient while waiting for the results and please try to limit your query results to a sensable amount so as not to overload the server. Anything below a 1 000 000 results should be OK to load but will take a long time to load.

Happy querying!

Available datasets

ptwp
The main dataset with all our data seperated into 4 graphs

  • data:TrafficData: 4039956 triples
  • data:PollutionData: 1338946 triples
  • data:WeatherData: 444548 triples
inf_ptw
A small sample set with only 1 month worth of data, but with inference enabled. 149015 triples all in the default graph.
